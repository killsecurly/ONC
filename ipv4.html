What's ONC for IPv4?

IPv4 (or Internet Protocol Version 4) is a internet address protocol that is most frequently used
around the world. Visually, it consists of a X.X.X.X address, each ranging from 0 to 255. (8-bit)
IPv4 and IPv6 are NOT cross compatible.


ONC for IPv4 is a IPv4 DNS Nameserver service.
Currently, ONC for IPv4 only supports static gateway public IPs.

How do I check what I'm using?

Click the button above and check the IPv4 and IPv6 fields.
Do you have a IPv4 number assigned? Then, your device and network are IPv4 compatible.
